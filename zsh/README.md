# ZSH configuration file

## Features
* Autocomplete
* Path expansion
* Shell Timeout
* Arrow navigation
* Stylized prompt
* Auto-CD
* Terminal Locking (Starts locked, timeouts 10 min) (requires vlock)
* Runs uptime every 60 seconds
* much more!
